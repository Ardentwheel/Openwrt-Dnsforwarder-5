Openwrt-dnsforwarder-5

Download this project as a .zip file Download this project as a tar.gz file Download this project as a .ipk file

DNS Cache Server with TCP ,UDP & GfwList

The ipk package whice can be downloaded is only for MT7620a CPU. If your machine are not MT7620a, you need to compile your own ipk package.

Default port is 5353, you can change it through ./files/dnsforwarder.config .

Install Dnsforwarder

wget http://matthew728960.github.io/Openwrt-Dnsforwarder-5/dnsforwarder-5-01-15-1_ramips_24kec.ipk;
opkg install dnsforwarder-5-01-15-1_ramips_24kec.ipk

Compile

(1) Copy Openwrt-Dnsforwarder-5 folder in to ./trunk/packages .	
(2) Put the dnsforwarder-5-01-15.tar.gz file to the dl folder.
(3) make package/Openwrt-Dnsforwarder-5/compile
(4) make package/Openwrt-Dnsforwarder-5/clean
